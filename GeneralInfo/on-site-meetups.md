# On-Site Meetups

ในโครงการนี้จะมีการพบปะกันแบบ on-site ทั้งหมด 2 รอบ (หากไม่สะดวกเข้าร่วมแบบ on-site สามารถเข้าร่วมบางกิจกรรมแบบออนไลน์ได้ ซึ่งไม่ส่งผลกระทบต่อการพิจารณาได้รับ certificate)

1. **งานปฐมนิเทศของโครงการ Brain Code Camp แบบ hybrid**

- **วันที่**: 2 ส.ค. 2569 (วันอาทิตย์)
- **เวลา**: 13:00-16:00
- **สถานที่**: ห้อง 1303 ชั้น 13 ตึกภูมิสิริมังคลานุสรณ์ โรงพยาบาลจุฬาลงกรณ์ (สำหรับผู้ที่ไม่สะดวกเข้าร่วมงานแบบ on-site สามารถเข้าร่วมแบบ online ผ่านช่องทางที่จะประกาศในภายหลัง)
- **การแต่งกาย**: ไม่มี dress code
- **จุดประสงค์**
  - ให้ข้อมูลและถามตอบข้อสงสัยเกี่ยวกับโครงการ
  - เปิดโอกาสให้ผู้เข้าร่วมกิจกรรมทั้งหมดได้ทำความรู้จักกัน
- **กิจกรรม**
  - แนะนำตัวผู้สอนและผู้เรียน
  - ให้ข้อมูลทั่วไปเกี่ยวกับโครงการ
  - ถามตอบข้อสงสัย
  - TA แต่ละกลุ่มแนะนำตัวและความเชี่ยวชาญ (กลุ่มละไม่เกิน 5 นาที) เพื่อให้ผู้เรียนสามารถแสดงความสนใจเข้ากลุ่มที่สนใจได้
  - พบปะผู้เรียนบางส่วนจากโครงการรุ่นก่อนหน้า และกิจกรรมอื่น ๆ (สำหรับผู้ที่เข้าร่วมงาน on-site เท่านั้น)

2. **Mid-Program On-site Meetup**

- **วันที่**: 20 ก.ย. 2569 (วันอาทิตย์)
- **เวลา**: 13:00-16:00
- **สถานที่**: ห้อง 1303 ชั้น 13 ตึกภูมิสิริมังคลานุสรณ์ โรงพยาบาลจุฬาลงกรณ์
- **การแต่งกาย**: ไม่มี dress code
- **จุดประสงค์**
  - เปิดโอกาสให้ผู้ร่วมกิจกรรมทั้งหมดได้ทำความรู้จักกันมากขึ้น
  - เปิดโอกาสให้ผู้เรียนได้พูดคุยแลกเปลี่ยนความรู้ ค้นหาและแลกเปลี่ยนไอเดียของโปรเจคที่สนใจ กับผู้สอนและผู้เรียนคนอื่น ซึ่งจะมีผลต่อการจัดกลุ่มใหม่ในช่วงท้ายของโครงการ
  - รับ feedbacks จากผู้เรียน เพื่อใช้ปรับปรุงกิจกรรม
- **หมายเหตุ**: หลังจากงานนี้จะมีการจัดกลุ่มใหม่ตามความสนใจของผู้เรียนและความเชี่ยวชาญของ TA โดยทีมงานจะให้ผู้เรียนทำแบบฟอร์มเลือกกลุ่มที่สนใจ และ TA ก็จะเลือกผู้เรียนด้วยเช่นกัน หลังจากนั้นทีมงานก็จะนำเอาข้อมูลที่ได้รับไปจัดกลุ่มใหม่ ให้ตรงกับความต้องการของทั้งผู้เรียนและ TA ให้ได้มากที่สุดเท่าที่จะเป็นไปได้ โดยจะประกาศกลุ่มใหม่ภายในวันที่ 30 ก.ย. 2569
